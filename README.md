# L2PHx Modern

Современная замена оригинального L2PHx (Delphi, ~2010) — перехватчик и анализатор пакетов Lineage 2 с веб-интерфейсом.

## Архитектура

```
Game Client ←→ [WinDivert NAT] ←→ L2 MITM Proxy ←→ Game Server
                                        ↕
                                   Web UI (WebSocket)
                                        ↕
                                   MCP (stdio)
```

## Реализованный функционал

### Ядро прокси (`l2_mcp_proxy.py`)
- **WinDivert 2.2** — перенаправление трафика на уровне ядра Windows (kernel-level NAT)
- **L2 криптография** — полная реализация Blowfish ECB + XOR stream cipher (4 независимых состояния)
- **Ertheia+ деобфускация** — обнаружение KeyInit по размеру пакета (≥153 байт), декодирование обфусцированных опкодов
- **Обход Frost/Teros** — WinDivert работает ниже уровня перехвата античитов
- **Intermediate протокол** — поддержка порта 17453 (очередь входа)
- **Автоопределение целевого сервера** — запись IP:port в файл для веб-интерфейса

### Веб-интерфейс (`l2phx.py`)
- **Реалтайм дашборд** — WebSocket обновления 25 FPS, тёмная тема
- **Таблица пакетов** — направление (C→S / S→C), опкод, имя, размер, время
- **Hex viewer** — просмотр сырых и расшифрованных данных пакета
- **Фильтрация** — по имени, опкоду, направлению; скрытие шумных пакетов
- **Парсер PacketsFreya.ini** — автоматический разбор структуры пакетов (Loop/Get)
- **Справочники** — ItemsID, NpcsId, SkillsId, ClassId, SysMsgId (RU + EN)

### Инжекция пакетов
- **Raw C2S / S2C** — отправка произвольных пакетов в hex
- **Чат** — отправка сообщений в игровой чат
- **Bypass** — обход ограничений клиента
- **Admin** — GM-команды
- **Enchant** — зачарование предметов
- **Use Item** — использование предметов
- **Flood** — массовая отправка пакетов

## Запуск

```bash
# Требуется запуск от администратора (для WinDivert)
python l2phx.py --divert

# С указанием целевого сервера
python l2phx.py --divert --target 5.63.128.2:7777

```

Веб-интерфейс откроется автоматически на `http://localhost:8877`

## Требования

- **Python 3.12+**
- **Windows** (WinDivert — только Windows)
- **Права администратора** (для загрузки WinDivert kernel driver)
- **aiohttp** (`pip install aiohttp`)

## Структура файлов

```

├── l2phx.py            # Веб-интерфейс + точка входа
├── l2_mcp_proxy.py     # Ядро: MITM прокси + крипто + MCP
├── l2_mcp_bridge.py    # Мост между веб-UI и MCP
├── WinDivert.dll       # WinDivert 2.2 библиотека
├── WinDivert64.sys     # WinDivert kernel driver
├── settings/           # Справочники опкодов и ID
│   ├── PacketsFreya.ini
│   ├── ItemsID.ini
│   ├── NpcsId.ini
│   ├── SkillsId.ini
│   └── ...
└── logs/               # Логи (создаётся автоматически)
```
